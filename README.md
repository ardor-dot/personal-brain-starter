# 나만의 지식체계 만들기 — 시작하기

이 폴더는 **대화만으로 나의 업무 지식이 쌓이는 개인 두뇌**입니다.
어려운 설정이 없습니다. 이 폴더 안에서 Claude Code를 켜고, 그냥 이야기하면 됩니다.

---

## 1. Claude Code 설치 (처음 한 번만)

컴퓨터의 **터미널**(Mac) 또는 **PowerShell**(Windows)을 열고, 아래 한 줄을 붙여넣은 뒤 Enter를 누릅니다.

- **Mac**
  ```
  curl -fsSL https://claude.ai/install.sh | bash
  ```
- **Windows**
  ```
  irm https://claude.ai/install.ps1 | iex
  ```

설치가 끝나면 `claude` 라고 입력하고, 안내에 따라 **Claude 계정(Pro 또는 Max)** 으로 로그인합니다.
자세한 내용은 공식 문서를 보세요: https://code.claude.com/docs

> 터미널이 처음이라 막히면, 컴퓨터에 **Claude 데스크톱 앱**(https://claude.ai/download)을 열어두고
> 막힌 화면을 캡처해 붙여넣으며 물어보면 한 단계씩 도와줍니다.

---

## 2. 나의 두뇌 시작하기 (3단계)

1. 이 폴더(`personal-kb`)를 컴퓨터에 내려받아 원하는 곳에 둡니다.
2. **이 폴더 안에서** Claude Code를 실행합니다.
   - Mac: 터미널에서 이 폴더로 이동한 뒤 `claude`
   - 잘 모르겠으면 Claude 앱에 "이 폴더에서 claude를 실행하고 싶어"라고 물어보세요.
3. 검은 화면이 뜨면 이렇게 말합니다:

   > **시작하자**

   그러면 몇 가지 질문으로 당신의 일과 관심사를 함께 정리하고, 그 내용이 `knowledge/` 폴더에 차곡차곡 저장됩니다.

---

## 3. 매일 이렇게 씁니다

말하는 대로 지식이 쌓입니다.

- "오늘 A사와 미팅했어. 다음 주까지 견적 보내기로 했어." → **결정·할 일로 기록**
- "김이사 연락처가 바뀌었어." → **인물 노드 갱신**
- "요즘 신제품 출시 준비가 제일 바빠." → **프로젝트로 정리**

찾을 때도 대화로:

- "이번 주에 내가 뭘 결정했지?"
- "A사 관련해서 아는 것 다 모아줘."
- "이번 주 정리해줘."

---

## 원칙

- 모든 기록은 **이 폴더 안에만** 저장됩니다. 밖으로 나가지 않습니다.
- 저장하기 전에 항상 **한 번 물어봅니다.** 원치 않으면 "그건 기록하지 마"라고 하면 됩니다.
- `knowledge/INDEX.md`를 열면 지금까지 쌓인 지식의 **목차**를 볼 수 있습니다.
- 이 폴더는 누구의 것도 아닌 **당신의 것**입니다. 당신의 일에 맞게 자유롭게 채워가세요.

막히면 언제든 Claude에게 **우리말로** 물어보세요. 그게 이 도구의 핵심입니다.
